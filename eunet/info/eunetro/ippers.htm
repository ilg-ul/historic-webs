PersonalEUnet

Descriere generala:

Este cea mai ieftina solutie de acces la serviciile Internet. Legatura se face pe linii comutate si se initiaza dinspre client catre EUnet. Odata stabilita legatura, calculatorul conectat este host in Internet, cu adresa de IP proprie.

Cerinte tehnice:

  • Calculator rulind DOS/Windows, si un pachet de comunicatii care implementeaza protocolul SLIP, sau un sistem UNIX care implementeaza protocolul SLIP.
  • Linie telefonica obisnuita si modem asincron V32.bis/V42.bis (max.14400+compresii).

Servicii oferite:

  • Conectivitate IP cu intreaga lume; routarea pachetelor se face preferential prin infrastructura EUnet; legaturile internationale se fac prin CIX (Commercial Internet Exchange), deci nu se aplica restrictii gen "NSFnet Acceptable Use Policy".
  • Inregistrarea in Domain Name Server a adresei dumneavoastra.
  • Acces la toate serviciile on-line (telnet, ftp, WWW, gopher, News, Archive, etc.) la cel mai bun pret posibil, taxarea facindu-se numai la volum, nediferentiat pe servicii sau distanta.
  • La cerere, putem furniza ultimele versiuni ale programelor de comunicatie disponibile ca free software.
  • Asistenta tehnica la instalare.
Mai consultati Tarife si procedura de inregistrare.
