telnet

Este un protocol folosit pentru conectarea la alte computere din Internet, oferind asa numitul Remote Login. Terminalul de pe care se lucreaza este conectat virtual la computerul gazda, in acest fel toate programele lansate se executa pe calculatorul gazda si nu pe cel local. Programul telnet care implemteaza acest protocol ofera o interfata compatibila cu terminalele vt100-vt320, iar programul tn3270 ofera o interfata compatibila cu terminalele IBM3270 folosite de calculatoarele mainframe. Programul rlogin ofera servicii asemanatoare cu telnet chiar daca se bazeaza pe standarde diferite. Telnet-ul este folosit in mod curent pentru a accesa diferite baze de date in mod interactiv.
