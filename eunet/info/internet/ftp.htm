ftp

Numele de ftp vine de la protocolul pe care il foloseste aplicatia: File Transfer Protocol. Acest program este folosit pentru transferul de fisiere intre masini conectate intr-o retea bazata pe protocolul TCP/IP. Acest protocol este implementat pe toate masinile care pot folosi TCP/IP-ul, de la marile mainframe-uri IBM pina la PC-uri, fiind suportat de aproape toti producatorii de sisteme de operare. El permite transferul fisierelor in format binar sau text.
