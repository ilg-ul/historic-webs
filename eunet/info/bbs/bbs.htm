Ce este un BBS?

Un BBS este un Bulletin Board System, adica un fel de avizier electronic, in care utilizatorii pot posta mesaje catre alti utilizatori sau catre grupuri de interes.

Din punct de vedere tehnic, este un computer dotat cu un modem conectat la reteaua telefonica. Utilizatorii se conecteaza cu un emulator de terminal sau cu un program special. Accesul se face pe bazu unui nume si a unei parole. De cele mai multe ori exista posibilitatea de a se face inregistrarea dinamic, chiar prin telefon, alteori este necesara interventia administratorului.

BBS-ul poate fi izolat sau poate fi conectat cu alte BBS-uri, comunicatia facindu-se la nivel de schimb de mesaje.
