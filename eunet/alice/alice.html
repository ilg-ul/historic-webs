Alice's Adventures in Wonderland

This HTML document was produced from the second Texinfo edition of a well-known book entitled Alice's Adventures in Wonderland written by Lewis Carroll, alias Charles Lutwidge Dodgson.

The original was first published in Great Britain in 1865. Alo contained herein, you'll find the sequel Through the Looking Glass and what Alice found there, that was first published in Great Britain in 1872.

Both of these original editions owe much to the illustrations by Sir John Tenniels. All of these 42 + 50 images are included in this edition of this online version.

Well, you should not really read the book here, you should download the compressed version from an archive and feed it into your local system. The default server of this project is ftp.Germany.EU.net, the directory is pub/books/carroll. There you find the PostScript, TeX DVI, GNU Info, HTML and ASCII versions; plus the complete Texinfo sources of this document (edn 2.0).

Note from EUnet Romania: if you connect to our server from abroad, please do not read this book from here. Our line is too slow and too expensive (for us!) to allow you to see all the nice pictures included in this book. Having much faster lines, please connect to http://www.Germany.EU.net.
