Things you may want to note:

  • A VALUE attribute to an INPUT tag of TYPE "text" specifies the default value of that text field.
  • A VALUE attribute to an INPUT tag of TYPE "checkbox" specifies the value that checkbox takes when it's on. If this attribute is not present, the default is "on".
  • A CHECKED attribute to a checkbox specifies that the checkbox is on by default.
  • Similar to how an INPUT tag of TYPE "submit" is a special query submission button, an INPUT tag of TYPE "reset" is a special form reset button. (If all of the input elements have no default values, this may be better termed a "clear" button; the label on the pushbutton included in the form is controlled by the VALUE attribute to that INPUT tag.)
