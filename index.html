Welcome to Historic Webs!

This web tries to preseve some very old web sites I was involved with.

EUnet

EUnet Romania was the first commercial Internet Service Provicer in Romania. These pages are among the first (if not really the first) web pages hosted on a Romanian HTTP server.

AOPA docs

The documents preserved in these pages reflect the early history of AOPA Romania, and the strugle to simplify the flight regulations (Romanian text).

Mobile Me

This was the first attempt to build the livius.net home page, and was stored on Apple Mobile Me servers. It never passed the experimental phase, but it is still nice to see it.
